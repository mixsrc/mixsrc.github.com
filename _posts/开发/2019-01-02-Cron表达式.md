## 格式

Cron 表达式指定任务在某个时间点或者周期性的执行，格式如下：

```
{秒} {分} {时} {日} {月} {周} [年]
```

## 字段可取值

| 字段 | 值                      |
| ---- | ----------------------- |
| 秒   | 0-59 , - * /            |
| 分   | 0-59 , - * /            |
| 时   | 0-23 , - * /            |
| 日   | 1-31 , - * / ? L W C    |
| 月   | 1-12 , - * /            |
| 周   | MON-SAT , - * / ? L C # |
| 年   | 2020-2099 , - * / 空    |

其中，

1-12 月也可以用字母表示：JAN, FEB, MAR, APR, MAY, JUN, JUL, AUG, SEP, OCT, NOV, DEC

一周的7天也可以用数字表示，1 代表星期天，依次为 SUN, MON, TUE, WED, THU, FRI, SAT

## 符号含义

| 符号 | 含义                                                                                                     |
| ---- | -------------------------------------------------------------------------------------------------------- |
| *    | 每，在秒字段表示每秒，在分字段表示每分，以此类推                                                         |
| -    | 范围，如 ```1-5```，在秒字段表示每分钟的第1到5秒，在分字段表示每小时的第1到5分钟，以此类推               |
| ,    | 指定值，如 ```1,5```，在秒字段表示每分钟的第1、第5秒，在分字段表示每小时的第1、第5分钟，以此类推         |
| /    | 起始和幅度，如 ```1/5```，在秒字段表示每分钟的第1秒开始，每5秒生效一次，即第1秒、6秒、11秒....，以此类推 |
| ?    | 仅用于日和周字段，表示不指定值，当其中一个有值时，另外一个需要设为?，日和周实际上是冲突的，不能同时指定  |
| L    | Last，最后的意思，在日字段表示一个月最后一天，在周字段表示一周最后一天，即周六                           |
| W    | Weekday，工作日的意思，工作日指周一到周五日                                                              |
| #    | 指定这个月的第几个周几，如 ```6#3``` 指这个月第3个周五，这里指定的时间可能不存在                         |
| C    | Calendar，不常用，网上的资料不明确且有矛盾之处，后续明确了再补                                           |

## 对 L、W 的补充说明

L：在周字段里和数字联合使用表示这个月的最后一个星期几，例如 ```6``` 表示这个月的最后一个星期五。

W：实际上指定的时最接近指定日的工作日。如 ```15W``` 指最接近这个月15号的工作日，如果15号是周六，那么指的是14号周五，如果15号是周日，那么指的是16号周一，需注意的是，值只会在当月计算，不会跨月，例如 ```1W```，如果1号是周六，那么指的是3号周一，不会返回上个月。另外，W字符仅能指明一天，不能是一个范围或列表。最后，可用 ```LW``` 来指定这个月的最后一个工作日。 

## 示例

| 示例                                 | 表达式                 |
| ------------------------------------ | ---------------------- |
| 从0分开始每5分钟                     | 0 0/5 * * * ？         |
| 每天的8点30分                        | 0 30 8 * * ?           |
| 工作日早上7点30分                    | 0 30 7 ? * MON-FRI     |
| 每个月的最后一天早上的10点           | 0 0 10 L * ?           |
| 每个月的最后一个工作日的晚上18点30分 | 0 30 18 LW * ?         |
| 2000年的10月份每个周六和周日12:34:56 | 56 34 12 ? 10 1,7 2000 |
